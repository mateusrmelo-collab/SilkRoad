# 📥 Silk Road - Media Downloader

Uma aplicação web desenvolvida com **Flask** para fins de aprendizado, que permite processar downloads de mídia a partir de URLs utilizando a biblioteca **yt-dlp**.

O projeto foi criado com foco em praticar desenvolvimento backend, integração com APIs/bibliotecas externas e deploy em ambiente de nuvem.

---

## 🌐 Demonstração

🔗 Acesse o projeto online:
https://silkroad-ns84.onrender.com/

---

## 🚀 Sobre o projeto

Este projeto é uma ferramenta simples de download de mídia que permite ao usuário inserir uma URL e escolher o tipo de arquivo desejado (vídeo ou áudio).

Ele foi desenvolvido como parte do processo de aprendizado em desenvolvimento web full stack.

---

## ⚙️ Funcionalidades

* Interface web simples e responsiva
* Seleção entre download de vídeo ou áudio
* Processamento de mídia via backend Flask
* Retorno do arquivo diretamente no navegador
* Integração com múltiplas plataformas suportadas pelo yt-dlp

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Flask
* yt-dlp
* HTML5
* CSS3
* JavaScript (Fetch API)
* Gunicorn (produção)

---

## ⚠️ Observações

Este projeto foi desenvolvido **exclusivamente para fins educacionais**, com o objetivo de praticar conceitos de backend, integração com bibliotecas externas e deploy em ambiente cloud.

Algumas plataformas podem impor limitações no processamento de downloads devido a restrições de rede e políticas anti-bot.
