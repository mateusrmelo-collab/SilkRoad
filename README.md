# 🎬 Silk Road Downloader

Um site feito com Flask que permite baixar vídeos e áudios de links (como YouTube) de forma simples e rápida.

---

## 🚀 Funcionalidades

- 📥 Download de vídeo (MP4)
- 🎧 Download de áudio (MP3)
- 🌙 Modo escuro (Dark Mode)
- ⚡ Interface simples e rápida
- 🧹 Limpeza automática de arquivos no servidor

---

## 🛠️ Tecnologias usadas

- Python
- Flask
- yt-dlp
- HTML5
- CSS3
- JavaScript
- Gunicorn (deploy)
